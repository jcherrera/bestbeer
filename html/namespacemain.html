def main.bb_search ( )

Run the beer search process.

Prompts the user for a search term. Searches for that term and provides user with results. User is then prompted to select from the results. The selected result info and user rating is displayed. If the user has yet to rate the beer in the result then they are prompted to do so. If the rating is high enough a sugesstion for another beer is displayed.
